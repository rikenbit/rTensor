# rTensor
Tools for Tensor Analysis and Decomposition

Installation
======
~~~~
git clone https://github.com/rikenbit/rTensor/
R CMD INSTALL rTensor
~~~~
or type the code below in the R console window
~~~~
library(devtools)
devtools::install_github("rikenbit/rTensor")
~~~~
